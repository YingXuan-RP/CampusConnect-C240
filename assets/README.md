# CampusConnect Portal - Image Assets

This folder contains all images used in the CampusConnect Portal.

## Required Images

### Hero Slideshow (3 images)
- **rp-campus-1.jpg** - Republic Polytechnic campus view (e.g., main building, entrance)
- **rp-campus-2.jpg** - RP campus facilities (e.g., library, study areas)
- **rp-campus-3.jpg** - RP student life (e.g., students on campus, activities)

Recommended size: **1920x500px** (landscape, high quality)

### Service Feature Backgrounds (5 images)
- **budget-makan.jpg** - Food/dining related image
- **facilities-booking.jpg** - Campus facilities/rooms
- **explore-ccas.jpg** - CCA activities/clubs
- **peer-support.jpg** - Students supporting each other
- **studyconnect.jpg** - Students studying together

Recommended size: **1200x300px** (wide landscape, high quality)

## Image Guidelines

- Use high-quality JPG images
- Ensure images are web-optimized (compressed but clear)
- Use relevant, professional images that represent Republic Polytechnic
- Images should have good contrast for text overlay readability
- All images should be royalty-free or properly licensed

## How to Add Images

1. Place your images in this `/assets/` folder
2. Use the exact filenames listed above
3. The portal will automatically display them
4. If images are not available, the green background color will show as fallback
